03 · Text-to-Speech

Nghe nội dung

Nhấn nút 🔊 cạnh mỗi đoạn văn để nghe. Double-click vào văn bản cũng kích hoạt. Nút 🔊 ở góc phải dưới đọc toàn bộ trang.

👆
Click nút 🔊 cạnh văn bản
Mỗi đoạn văn quan trọng có nút loa nhỏ ở cuối. Nhấn để nghe giọng đọc bằng ngôn ngữ hiện tại.
✌️
Double-click để nghe
Nhấn đúp vào bất kỳ đoạn văn có thể nghe để kích hoạt giọng đọc — không cần tìm nút bấm.
🌐
Giọng đổi theo ngôn ngữ
Khi bạn chuyển sang tiếng Anh, Nhật, Hàn... giọng đọc sẽ tự động dùng đúng ngữ âm. Không đọc sai tiếng.
📱
Zero dependency
Web Speech API là tính năng sẵn có trên trình duyệt — không cần server, không cần API key, không tốn thêm chi phí.

♿ Accessibility — Tính năng này hỗ trợ:
• Người dùng quốc tế (7 ngôn ngữ, tự nhận diện trình duyệt)
• Người khiếm thị / mắt kém (Web Speech API, không cần cài thêm)
• Người không biết chữ / dân tộc thiểu số (đọc to giao diện theo tiếng bản địa)
• Lưu lựa chọn ngôn ngữ — không cần chọn lại mỗi lần
// Cách tích hợp vào file HTML bất kỳ
<script src="hlrace-i18n.js"></script>
// Đánh dấu text cần dịch:
<span data-i18n="reg.title">Đăng ký</span>
// Đánh dấu text cần có nút loa:
<p data-tts>Nội dung muốn đọc to</p>
// Đổi ngôn ngữ bằng JS:
HLi18n.setLang('en');
// Đọc to một đoạn text:
HLi18n.speak('Chào mừng đến với HLRace');
